❓ Frequently Asked Questions

What is this platform?

This is a crypto project discovery and escrow system built for testing and demonstration.

How does escrow work?

The user submits a request, which is reviewed by the admin before funds are securely handled off-platform.

Can I add my own project?

Yes! Go to the Submit page and fill the form.

Is this live on blockchain?

No — this is a frontend simulation. Real functionality can be added with smart contracts or backend logic later.
